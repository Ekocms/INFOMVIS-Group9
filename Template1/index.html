How to read “Projects by NbS Type”

What this shows: the number of projects that include each NbS Type in the current selection.

Why other bars still appear: projects can belong to multiple NbS Types. Selecting one type filters the projects, but those projects may still count toward other types too.

How to use it: click a bar (or its label) to filter all views by that NbS Type. Click again to clear.
